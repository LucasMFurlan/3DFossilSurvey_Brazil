# 3DFossilSurvey_Brazil
Database containing orthophoto datasets and point cloud data of fossil specimens collected from a submerged outcrop in the Passa Cinco River, Ipeúna, Brazil.

Submerged Fossil Photogrammetry Database – Passa Cinco River, Brazil

This repository contains orthophoto datasets and dense point cloud data of fossil specimens collected from a submerged rock outcrop in the Passa Cinco River, Ipeúna, São Paulo State, Brazil.
The data correspond to all specimens identified during the first phase of the project using integrated aerial and underwater photogrammetry.

Repository Structure

Specimens A to I:
Orthophotos generated from drone-based aerial photogrammetry.
Raw dense point cloud data derived from underwater image matching.

Study Area
Passa Cinco River, Ipeúna, São Paulo State, Brazil
Shallow submerged fossiliferous outcrop
